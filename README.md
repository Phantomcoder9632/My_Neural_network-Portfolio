## ✨ Core Features

🚀 **Immersive 3D Experience**
> Featuring a mathematically driven "Neural Network" tunnel built with `Three.js` and `@react-three/fiber` that dynamically reacts to the user's native scroll behavior.

⚡ **Real-Time Data Execution**
> Highlights agentic Kappa Architecture (Pathway), zero-latency WebSockets (FastAPI), and LLM deployments (Llama 3, Gemini, Zephyr-7B).

🎨 **Glassmorphism & Neon UI**
> Built strictly with Framer Motion and raw CSS to deliver stunning, hardware-accelerated glowing card grids and frosted-glass navigation elements.

## 🚀 Prominent Engineering Showcases

### 🏆 WikiSentinel: Autonomous Vandalism Detection
* **2nd Prize Winner** / Agentic RAG / Kappa Architecture
A real-time processing pipeline treating massive text flows as "motion". Uses **Pathway**, **Llama 3**, and **Gemini Vision** to cross-verify live Wikipedia edits instantly against rulebooks, slashing false-positive moderation by 40%.

### 📡 Velloc: AI Gesture Media Engine
* Computer Vision / Edge AI / WebSockets
A touchless `<10ms` interface using **MediaPipe** extracting 63-point 3D spatial hand landmarks and a custom **Scikit-Learn** Random Forest mapping to 16 native desktop gestures.

### 🧬 Clinical Speech Processing Pipeline
* Displace 2026 Challenge / AI Research
Built complex speaker diarization sliding-window architectures with `ECAPA-TDNN` while fine-tuning `Whisper Large v3` and cascaded `Zephyr-7B` for summarization.

## 🖥️ Local Initialization

To run the Neural Core environment on your local machine:

```bash
# Clone the repository
git clone https://github.com/Phantomcoder9632/My_Neural_network-Portfolio.git

# Enter the directory
cd My_Neural_network-Portfolio

# Install dependencies matrix
npm install

# Ignite the dev server
npm run dev
```
